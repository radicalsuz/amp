Install instructions:

1. edit Connections/config.php to set the database, path and other settings.
2. edit system/spaw/config/spaw_control.config.php
3. change the chmod settings on the following directories to 777
	/img
	/img/pics
	/img/thumb
	/img/orgiginal
	/downloads
	/system/mailblast/blast.log
	/system/mailblast/smtp_client.log
4. log into the administration system at /system (admin password user:admin password:changme) 
5. change the admin password under system settings -system users
